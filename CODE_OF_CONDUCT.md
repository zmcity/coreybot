# 贡献者行为准则

## 我们的承诺

为营造开放、友善的社区环境，我们作为贡献者与维护者承诺：无论年龄、体型、残障与否、民族、性别、经验水平、国籍、个人形象、种族、宗教或性取向，参与本项目的每一个人都能免于骚扰。

## 我们的准则

- 使用友善、包容的语言；尊重不同观点与经验。
- 优雅地接受建设性批评；以社区利益为重。
- 不得使用性化语言或图像、人身攻击、公开或私下骚扰。

## 执行

若遇到违反行为，请通过 Issue 或邮件联系维护者。本准则改编自 [Contributor Covenant](https://www.contributor-covenant.org/)。
